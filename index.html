🎯 점검 준비 핵심 사항

🚨 핵심 포인트

  • 6/12 SQ평가 지적사항: 경신에 내부 전달된 상황 - 개선 필수
  • 준비 범위: 두 품번 원재료→출하 + 주변 설비 문서
  • 대응 구조: 각 팀/파트가 직접 대응 (팀장 1명으로 3명 대응 불가)
  • 점검 강도: 3명 동시 점검 = 12시간 효과

🔄 CAPD 구조 기반 준비 실행

⚡ 선조치(C) - 즉시실행

  • 생산팀 (추홍근 팀장): 6/26까지 두 품번 스케줄 반영 (최소 D-1)
  • 설비팀: 전공정 설비/ 점검 Check Sheet (6/12 이후 Stop 상태 해결)
  • 사출팀: 설비/공무쪽으로 내용 전달 필수

📋 계획(P) - 문서준비 체계

  • 1단계: 원재료→출하 전체 관리문서 완비
  • 2단계: 앞뒤 좌우 설비의 공정 비치 문서
  • 3단계: 현장게시물 최신 상태 확인
  • 핵심: 6/12 SQ평가 문제점 위주 공정 준비

👥 실행(D) - 당일대응 시나리오

  • 현실: 품질팀장 1명이 3명 점검자 동시 대응 불가능
  • 해결책: 각 팀장/파트장이 담당영역 직접 대응
  • 필수조건: 관련 지침들을 꼭!! 인지하고 대응
  • 당일 원칙: 주간 Set-up 최소화
